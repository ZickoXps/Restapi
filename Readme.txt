Install

Requerimientos
#AndroidStudio
#Xampp
#####################################################################################
1. Importar la base de datos "restapi.sql" a phpmyadmin
2. Copiar la carpeta rest dentro de la ruta C:\xampp\htdocs
3. Descomprimir el zip "RestApi.zip" y abrir el proyecto en Android Studio
4. En la pestana Build dentro de Android Studio dar click en Make Proyect o Make Module (para compilar el codigo)
5. Java > com.byazbrian.restapi > LoginGen > Linea 30 cambiar ip por ipv4 (Ipconfig en windows y ifconfig linux verla ipv4)
6. Realizar este mismo metodo en  Java > com.byazbrian.restapi > Lobby2 > Linea 31
7. Ejecutar el proyecto.
#####################################################################################
